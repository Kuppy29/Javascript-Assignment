Click me event </button>

}

2. JavaScript onkeyup event:

This event is a keyboard event and executes instructions whenever a key is released after pressing.

Examples :

In this example, we will change the color by pressing UP arrow key


<h4>The input box will change color when UP arrow key is pressed</h>
<inut id="bg" onkeyup="changeBackground()" placeholder="write something" style="color:#fff">
<script>
var a=0;
var b=0;
var c=0;
function changeBackground() {
var x=document.getElementById('bg');
x.style.backgroundColor='rgb('+a+', '+b+', '+c+')';
a+=100;
b+=a+50;
c+=b+70;
if(a>255) a=a-b;
if(b>255) b=a;
if(c>255) c=b;
}
</script>

The input box will change color when UP arrow key is pressed

This event corresponds to hovering the mouse pointer over the element and its children, to which it is bound to


Example :

In this example, we will make the box vanish when the mouse will be hovered on it


<div id="hover" onmouseover="hov()" style="background-color:green; height:200px; width:200px;"> </div> <script> function hov() { var e=document.getElementById('hover'); e.style.display='none'; } </script>
