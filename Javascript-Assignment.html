br

--> All three looping structures serve different purposes and can be used based on the specific requirements of your JavaScript code. The for loop is commonly used when you know the number of iterations in advance, the while loop is used when the number of iterations is uncertain, and the do-while loop is suitable when you want to execute the loop body at least once before checking the condition.


1. for loop :- The for loop is one of the most commonly used looping structures in JavaScript. It allows you to specify an initialization, a condition, and an iteration statement.


Syntax :-

javascriptCopy codefor (initialization; condition; iteration) {
// Code to be executed in each iteration br }


Example :-

}



2. while loop :- The while loop repeats a block of code as long as a specified condition evaluates to true.


Syntax :-

javascriptCopy codewhile (condition) {
// Code to be executed in each iteration br }


Example :-

javascriptCopy codelet i = 0;
while (i < 5) {
console.log("Iteration: " + i);
i++;
}


3. do-while loop :- Similar to the while loop, the do-while loop also repeats a block of code as long as a specified condition evaluates to true. However, the do-while loop executes the code block at least once before checking the condition.


Syntax :-

javascriptCopy codedo {
// Code to be executed in each iteration
} while (condition);


Example :-

javascriptCopy codelet i = 0;
do {
console.log("Iteration: " + i);

1. Alert box

2. Confirm box

Alert box : It is used when a warning message is needed to be produced. When the alert box is displayed to the user, the user needs to press ok and proceed.


Syntax :

alert("your Alert here")


Example :

}

Syntax :

confirm("your query here")


Example :

var x;
if (confirm("Press a button!") == true) {

}

function Prompt() {
